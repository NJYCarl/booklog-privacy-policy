# 藏书记技术支持

更新日期：2026-05-31

感谢使用藏书记。藏书记是一款本地优先的阅读记录 App，用于管理个人书库、阅读会话、书摘书评、年度统计、Widget、导入导出和 iCloud 快照同步。

## 联系方式

如果你在使用中遇到问题，或希望反馈建议，请通过 App Store 产品页提供的支持渠道联系开发者。

反馈问题时，建议提供以下信息：

- App 版本号。
- iOS / iPadOS 版本。
- 设备型号。
- 问题发生前的操作步骤。
- 是否开启 iCloud、Widget、相机、相册等相关功能。
- 如有错误提示，请附上提示内容。

## 常见问题

### 1. 不登录可以使用吗？

可以。藏书记的核心功能不需要登录。书库、阅读记录、统计、导出和分享默认保存在本机。

### 2. 本机账户是什么？

本机账户只用于当前设备上的本地登录状态和显示名。藏书记不要求使用开发者运营的远端账号服务，也不会把本机账户资料上传到开发者服务器。

### 3. 如何删除账户？

你可以在 App 的设置页删除本机账户。删除本机账户会移除此设备上的账户资料和密码凭据，并退出当前登录，但不会自动删除本地书库、阅读记录、统计、封面或导出文件。

### 4. 如何删除阅读数据？

你可以在 App 内删除具体书籍和相关记录；也可以卸载 App 来移除本机 App 数据。已经手动上传到 iCloud 的快照由你的 Apple ID 和 iCloud 控制。

### 5. iCloud 同步如何工作？

藏书记的 iCloud 同步是手动快照同步。你可以在设置页检查 iCloud 状态、上传当前书库快照、合并最新快照。数据写入你自己的 Apple ID 私有 iCloud 数据库，开发者不会直接读取你的 iCloud 私有数据库。

### 6. Widget 没有更新怎么办？

请先打开藏书记并确认书库中已有阅读记录，然后返回主屏幕等待 Widget 刷新。如果仍未更新，可以尝试移除并重新添加 Widget，或重启设备后再试。

### 7. ISBN 扫描或拍照无法使用怎么办？

请在系统设置中允许藏书记使用相机。无相机设备、模拟器或拒绝相机权限时，仍可手动输入 ISBN 和书籍信息，也可以从相册选择封面。

### 8. 联网搜索失败怎么办？

联网搜索是可选增强功能，只在你主动搜索书名、作者或 ISBN 时使用。如果国家图书馆或 Open Library 搜索暂时不可用，你仍可手动录入完整书籍信息。

### 9. 如何导出数据？

你可以在设置页使用 JSON / Markdown 导出功能。导出文件由你主动生成，并由你自行保存或分享。

# Cang Shu Ji Support

Last updated: 2026-05-31

Thank you for using Cang Shu Ji. Cang Shu Ji is a local-first reading log app for managing a personal library, reading sessions, excerpts, reviews, annual statistics, widgets, import/export, and optional iCloud snapshot sync.

## Contact

For support requests or feedback, contact the developer through the support channel listed on the App Store product page.

When reporting an issue, please include:

- App version.
- iOS / iPadOS version.
- Device model.
- Steps taken before the issue occurred.
- Whether iCloud, widgets, camera, or Photos features were involved.
- Any visible error message.

## FAQ

### 1. Can I use the app without signing in?

Yes. Core features do not require sign-in. Your library, reading records, statistics, exports, and share content are stored locally by default.

### 2. What is a local account?

A local account is only used for sign-in state and display name on the current device. Cang Shu Ji does not require a developer-operated remote account service and does not upload local account data to a developer server.

### 3. How do I delete my account?

You can delete the local account in Settings. Deleting a local account removes the local profile and password credential from this device and signs you out, but it does not automatically delete your library, reading records, statistics, cover images, or export files.

### 4. How do I delete reading data?

You can delete individual books and records inside the app, or uninstall the app to remove local app data. iCloud snapshots that you manually upload are controlled by your Apple ID and iCloud settings.

### 5. How does iCloud sync work?

Cang Shu Ji uses manual iCloud snapshot sync. In Settings, you can check iCloud status, upload the current library snapshot, and merge the latest snapshot. The data is stored in your own Apple ID private iCloud database, and the developer does not directly access it.

### 6. What should I do if the Widget does not update?

Open Cang Shu Ji and confirm that your library has reading records, then return to the Home Screen and wait for the Widget to refresh. If it still does not update, remove and add the Widget again, or restart the device.

### 7. What should I do if ISBN scanning or camera capture does not work?

Allow Cang Shu Ji to use the camera in system Settings. If the device has no camera, if you are using a simulator, or if camera permission is denied, you can still manually enter ISBN and book information or choose a cover from Photos.

### 8. What should I do if online search fails?

Online book search is optional and only runs when you search by title, author, or ISBN. If National Library of China search or Open Library search is temporarily unavailable, you can still manually enter complete book information.

### 9. How do I export data?

Use the JSON / Markdown export tools in Settings. Export files are generated only when you request them, and you decide where to save or share them.
